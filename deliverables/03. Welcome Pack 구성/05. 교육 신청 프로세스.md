# 3. 교육 신청 프로세스



## 3.1. 개요

체계적인 교육 관리 및 수행으로 Container를 위한 개발 방법을 교육하여 컨테이너 기반 클러스터 환경(K8S)에서 프로젝트를 진행에 대한 이해도를 높이고, 효율적인 활용이 될수 있도록 합니다.



## 3.2. 교육 커리큘럼

| --------- | ------------------------------------- |
| 교육 내용 | Docker 개요 ( Docker Network 사용법 ) |
|           | Spring Boot를 활용한 개발             |
|           | Git 서버 사용법                       |
|           | CI/CD 간략 소개                       |
